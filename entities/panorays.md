# Panorays

## Overview

Panorays is a global TPRM platform focused on continuous vendor security assessments. Growing presence in ANZ.

## Products

- Continuous third-party security monitoring
- Automated security questionnaires
- Threat intelligence integration
- Compliance reporting

## Market Position

- Continuous monitoring vs point-in-time assessments
- Competitor to UpGuard, Black Kite, BitSight

## ANZ Relevance

- Growing ANZ presence
- Useful for [[apra-cps-230]] continuous monitoring requirements
- Addresses the 71% gap (companies that assess vendors without security practices — InConsult/IIA research)
