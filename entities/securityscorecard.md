# SecurityScorecard

## Overview

SecurityScorecard is a global security ratings vendor providing algorithmically-derived cyber security scores for organisations and their vendors. One of the largest security ratings providers alongside BitSight.

## Products

### Security Ratings
- Continuous, externally-observable security ratings
- Historical rating tracking
- Peer comparison and benchmarking
- Threat actor alignment (tracks which threat actors target certain industries/geographies)

### TPRM Integration
- Vendor monitoring with automated alerts on rating changes
- Integrates with TPRM workflows (ProcessUnity, ServiceNow, RSA Archer)
- Questionnaire and assessment modules

## Market Position

- Primary competitors: BitSight, Black Kite
- Often used as input to TPRM programs rather than full TPRM platform
- Strong in enterprise due to brand recognition
